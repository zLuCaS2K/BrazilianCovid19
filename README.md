## 📋 Funcionalidades
### Documentação
- [ ] Dicas de prevenção.
- [ ] Barra de navegação inferior.
- [ ] Visão geral da situação de casos confirmados e óbitos no Brasil.
- [ ] Listagem da situação de cada estado do país.
- [ ] Mapa de calor.
- [ ] Banco de dados usando room para uso offiline.
